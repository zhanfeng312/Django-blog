### 2、pip -V 确认 pip版本

建议升级最新版本，用下面的命令升级pip

python -m pip install --upgrade pip

如果遇到错误：AttributeError: 'NoneType' object has no attribute 'bytes'

解决方法：easy_install -U pip

### 3、requirements.txt
